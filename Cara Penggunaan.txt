Silahkan ikuti panduan berikut demi kelancaran proses penggunaan aplikasi:

#### Peanduan Menjalankan Aplikasi ####
1. Jalankan aplikasi degan kode "python main.py" pada terminal vscode sesuai lokasi 
    penyimpanan folder. sebagai contoh "D:\CATATAN KULIAH\Semester 6\Pemrograman Visual\Project UAS\realproject> python main.py"
2. ketikkan perintah "python main.py" (pada langkah nomor 1) untuk mulai menjalankan aplikasi.
3. jika terdapat error silahkan melakukan download package yang dibutuhkan sesuai yang tertera 
    pada pesan error (contoh: pip install pillow)
4. jika terjadi masalah (error terus berlanjut) silahkan hubungi pembuat aplikasi


#### Panduan Pembuatan Akun Admin ####
Akun default untuk admin (email: admin@gmail.com - password: 12345)

Jika pengguna ingin membuat akun admin baru, silahkan ikuti langkah-langkah berikut:
1. Silahkan daftar seperti biasa pada halaman registrasi
2. Kami menyarankan pengguna menggunakan aplikasi tableplus untuk mengakses databse (mahasiswa.db)
3. Setelah melakukan registrasi dan mengakses database beserta tabel menggunakan aplikasi tableplus,
    silahkan ubah atribut level pada tabel users di aplikasi tableplus dari siswa menjadi admin.
    atau bisa lansung ubah dari kode editor masing-masing jika memungkinkan mengubah atribut pada tabel users
    dalam database mahasiswa.
4. Atribut yang diubah merupakan atribut yang ada pada database mahasiswa.db kemudian tabel users (bukan isi kode dari database.py)
5. Jika atributnya sudah diubah maka akun tersebut secara otomatis menjadi akun admin,
    dan secara otomatis akan mengarahkan ke dahsboard/halaman admin.
