Please follow these instructions to ensure a smooth application usage:

#### Application Run Guide ####
1. Install (mandatory) the requirements for reportlab, customtkinter, and tkcalendar (e.g., pip install reportlab).

2. Run the application with the code "python main.py" in the VS Code terminal according to the folder location.

    For example, "D:\LECTURE NOTES\Semester 6\Visual Programming\UAS Project\realproject> python main.py".

3. Type the command "python main.py" (in step 1) to start the application.

4. Wait for the application to open (this can take a while depending on your laptop's specifications).

5. If you encounter an error, please download the required package as indicated in the error message (e.g., pip install tkcalendar).

6. If you encounter problems (the error persists), please contact the application developer.


#### Admin Account Creation Guide ####
Default account for admin (email: admin@gmail.com - password: 12345)

If a user wants to create a new admin account, please follow these steps:
1. Register as usual on the registration page.

2. We recommend using the TablePlus application to access the database (mahasiswa.db).

3. After registering and accessing the database and tables using the TablePlus application,
please change the level attribute in the users table in the TablePlus application from student to admin.

You can also change the attribute directly from your code editor if possible by changing the attribute in the users table in the student database.

4. The attributes to be changed are those in the student.db database and the users table (not the code content in database.py).

5. Once the attribute has been changed, the account automatically becomes an admin account,
and will automatically redirect to the dashboard/admin page.
